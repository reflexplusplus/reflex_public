ASIO Support (Optional External SDK Dependency)
-----------------------------------------------
This distribution does not include Steinberg ASIO SDK files.

On Windows, the prebuilt static libraries include a 'ReflexTargetAudioAppASIO'
variant built against the Steinberg ASIO SDK headers.

Using this variant requires the Steinberg ASIO SDK to be obtained
separately, and the required ASIO source file(s) to be added to your
project for static linking.

Use of the Steinberg ASIO SDK is subject to Steinberg's applicable
licensing terms.

ASIO is a trademark of Steinberg Media Technologies GmbH,
registered in Europe and other countries.
