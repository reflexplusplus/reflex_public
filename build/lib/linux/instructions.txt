Linux build instructions

Prerequisites (Debian/Ubuntu)

sudo apt update
sudo apt install -y build-essential pkg-config libcurl4-openssl-dev libwayland-dev wayland-protocols libegl1-mesa-dev libgles2-mesa-dev

Build location

cd /mnt/d/devt/reflex/build/lib/linux

Common commands

make clean
make debug
make release
make

Build individual archives

Debug
make ../../../bin/lib/linux/Debug/libReflexCommon.a
make ../../../bin/lib/linux/Debug/libReflexTargetConsole.a
make ../../../bin/lib/linux/Debug/libReflexTargetApp.a

Release
make ../../../bin/lib/linux/Release/libReflexCommon.a
make ../../../bin/lib/linux/Release/libReflexTargetConsole.a
make ../../../bin/lib/linux/Release/libReflexTargetApp.a

Output folders

../../../bin/lib/linux/Debug
../../../bin/lib/linux/Release

Notes

ReflexTargetConsole is the headless Linux console/system target.
ReflexTargetApp is the Linux desktop/UI target.
This Makefile builds static libraries only; it does not build a final executable.
