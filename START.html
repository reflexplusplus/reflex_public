Get Started

Welcome to Reflex++

This page points you to the best places to start when exploring the framework so you can get started efficiently.

1. Build Reflex

  • If you installed Reflex using the Reflex Installer, pre-built libraries are already included.
  • If you cloned reflex_public from GitHub, run install.bat to download the required pre-built libraries and dependencies.
  • Users with a commercial licence with full source code can find build scripts and project files under: build/lib/

2. Documentation

  • An interactive ReflexDocumentation app is available at bin/tools.
  • We highly recommend you read the Fundamentals and Key Concepts section in the root Reflex guide.

3. Create a Project

  • Use the Reflex CLI (bin/tools) to generate a project from a template using the create command.
  • Alternatively, use the Project Creator app for a convenient UI front end.
  • Template projects include setup code and an initial structure so you can start developing immediately.

4. Example Project

For a practical working example, see the examples/Notes (Cpp App) project. It demonstrates the recommended project structure, separation between UI and application logic, and typical usage patterns for Reflex modules in a small but complete project.

5. Agents

Documentation is also exported in Markdown format under agents/ for use with AI tools and coding agents. We additionally recommend copying the include/reflex directory into your project so agents have direct access to the Reflex API and headers.
